# Noctyra Launcher

A lightweight open source launcher built with Rust, Tauri, and TypeScript.

Created by yZarickZ.

## Goals

- Linux support
- Windows support
- Lightweight
- Customizable
- Open source

## Technologies

- Rust
- Tauri
- TypeScript
